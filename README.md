# DDT with PyTorch

Third-party PyTorch implementation of DDT(Unsupervised object discovery and co-localization by deep descriptor transformation) [1]. Please cite this paper if you use this code.

## Dependencies
- pytorch
- torchvision
- cv2

## Demo

```
python ddt.py
```
